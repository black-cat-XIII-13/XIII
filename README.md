البوابة الشخصية والمحفظة الإلكترونية (Portfolio) لـ **BLACK CAT**. مساحة رقمية تجمع بين غموض القطط السوداء، سحر الرقم **13**، والتأملات الفلسفية العميقة المربوطة بحكمة القرآن الكريم في إيجاز بليغ.

## 🌟 الميزات (Features)

- **تصميم ليلي غامض (Dark Theme):** واجهة مستخدم تعكس الهوية الغامضة والأنيقة.
- **تأثيرات بصرية تفاعلية:**
  - خلفية تفاعلية للنجوم (Starfield Canvas) تتحرك مع المستخدم.
  - توهج مخصص يتبع مؤشر الماوس (Cursor Glow).
- **تجربة صوتية محيطية:** مشغل موسيقى مدمج لموسيقى "العراب" (The Godfather) لإضافة طابع مهيب أثناء التصفح.
- **أقسام متدرجة فلسفياً:**
  - **البداية (Hero):** مقدمة قوية مع الطباعة الأنيقة.
  - **الهوية (Identity):** تعريف بالشخصية وعلاقتها بالرقم 13.
  - **التأملات (Philosophy):** دمج بين الفلسفة وإعجاز القرآن.
  - **١٣ حقيقة (Reflections):** عرض مميز لمعلومات وتأملات سريعة.
  - **صندوق الصمت (Contact):** واجهة تواصل مبتكرة و "غامضة".
- **طباعة احترافية (Typography):** استخدام خطوط Google المتناسقة:
  - `Amiri`: للنصوص الفلسفية والقرآنية.
  - `Cairo`: لعناصر واجهة المستخدم.
  - `Cinzel`: للعناوين الإنجليزية والأرقام.

## 🛠️ التقنيات المستخدمة (Tech Stack)

تم بناء هذا المشروع بالاعتماد كلياً على التقنيات الأساسية للويب (Vanilla Web Technologies) لضمان أعلى أداء وتحكم كامل في التفاصيل:

- **HTML5:** لبناء الهيكل الدلالي للصفحة.
- **CSS3:** واجهة مصممة بالكامل من الصفر (Custom Design System)، باستخدام المتغيرات (Variables)، وتأثيرات الزجاج (Glassmorphism)، والرسوم المتحركة.
- **JavaScript (ES6):** لإدارة حركة النجوم (Canvas API)، تأثيرات الماوس، مشغل الصوتيات المحيطي، والتمرير السلس.

## 🚀 كيفية التشغيل (How to Run)

المشروع لا يتطلب أي خوادم أو إعدادات معقدة.
1. قم بتحميل أو استنساخ (Clone) المستودع:
   ```bash
   git clone https://github.com/cblackcat13-debug/XIII.git
   ```
2. افتح المجلد، وانقر نقراً مزدوجاً على ملف `index.html` لتشغيله في أي متصفح ويب حديث.
